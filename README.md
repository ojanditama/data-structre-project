# SISTEM ADMINISTRASI RUMAH SAKIT (POLIKLINIK & DOKTER) 

# Deskripsi

Sistem Administrasi Rumah Sakit (Poliklinik & Dokter) adalah sistem yang memodelkan hubungan antara poliklinik dan dokter yang bekerja di dalamnya. Setiap poliklinik memiliki beberapa dokter dengan spesialisasi tertentu, jam praktik, dan lama pengalaman kerja. Sistem mendukung pencarian poliklinik, penelusuran dokter berdasarkan spesialis, hingga perhitungan jumlah dokter di setiap poliklinik. Selain itu, aplikasi dapat menampilkan polikliinik dengan jumlah dokter terbanyak sehingga dapat digunakan untuk analisis internal. Proyek ini menekankan kemampuan dalam merancang struktur data hierarkis yang representatif untuk data layanan kesehatan.
